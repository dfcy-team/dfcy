# 阶段2验收检查清单

## API接入验收

- [ ] 连接器仅通过后端调用，Mock/沙箱优先，未连接真实平台。
- [ ] API Key/API Secret 未明文入库、未进入 Git、未出现在日志或前端。
- [ ] 请求签名、限流、超时、重试、幂等、webhook 验签和数据清洗具备测试证据。
- [ ] 失败告警、脱敏错误和灰度/回退策略已验证。

## 状态机验收

- [ ] 新品、正常销售、滞销、清仓、停售、归档状态及其流转规则可验证。
- [ ] API、RPA 回读和人工标记来源的优先级可验证。
- [ ] 自动建议不直接触发高风险业务动作，人工确认记录可审计。

## 财务对账验收

- [ ] 平台账单、取款记录、银行到账和手工导入仅使用脱敏或 Mock 数据。
- [ ] 自动匹配、差异标记和人工处理流程可验证。
- [ ] 财务独立授权生效，普通 internal、external、rpa 用户被拒绝。

## 供应商绩效验收

- [ ] 所有查询按 `tenant_id + supplier_id` 过滤。
- [ ] 交付及时率、逾期率、异常率、反馈及时性、出货准确率和评分规则可解释。
- [ ] 供应商只能查看自身绩效，内部汇总受权限控制。

## RPA稳定性验收

- [ ] 页面变化检测、失败截图、日志留存、重试上限和失败转人工可验证。
- [ ] 同账号高风险任务串行，complete/fail 仅回写结果。
- [ ] RPA 不访问 finance/internal/admin，不直连数据层，不连接真实 BigSeller。

## CI/CD验收

- [ ] 后端检查与测试、前端构建、契约测试、RPA JSON 校验、依赖扫描和密钥扫描有可复现记录。
- [ ] 流水线使用非生产配置，未携带真实凭据或业务数据。

## 安全扫描验收

- [ ] 未提交 `.env`、私钥、证书、真实 Token、Cookie、Session、API Key 或数据库密码。
- [ ] 日志、截图、测试夹具和构建产物不含真实平台、财务、供应商或订单数据。

## 真实平台接入专项评审验收

- [ ] 已完成凭据、网络、签名、回调、审计、灰度、回退和紧急停止方案审查。
- [ ] 已取得安全负责人和业务负责人书面批准。
- [ ] 未通过专项评审时，真实平台与高风险自动化保持禁用。
