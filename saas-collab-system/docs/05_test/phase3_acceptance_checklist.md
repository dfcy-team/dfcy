# 阶段3验收清单

## 数据与指标

- [ ] 指标口径、来源、tenant、质量规则和版本均可追踪。
- [ ] 跨 tenant、过期、缺失或质量失败数据不能进入正式汇总。
- [ ] BI、库存、生命周期、供应商和财务数据范围符合后端权限。

## 建议与预警

- [ ] 库存预警、补货建议、生命周期建议和经营异常只产生建议。
- [ ] 前端不能直接创建采购订单、改变生命周期或触发真实 RPA。
- [ ] 人工确认、忽略和阈值调整均有审计证据。

## 安全与平台

- [ ] 无真实 `.env`、凭据、平台 URL、银行/支付数据或业务数据进入仓库。
- [ ] 真实平台、真实高风险自动化和资金动作保持禁用。
- [ ] 供应商不访问 internal/finance；RPA 不访问 finance/internal/admin。

## 工程门禁

- [ ] Django check、迁移、pytest、前端构建、Docker 配置、RPA JSON 和安全扫描有可复现结果。
- [ ] 前端测试脚本与页面冒烟测试覆盖新增高频页面。
- [ ] 未完成 API 标记 pending/mock，未伪造 connected。

## 配置与导出

- [ ] 配置版本、生效时间、审批、变更记录和回滚机制可验证。
- [ ] 财务字段脱敏，报表导出具有权限和审计记录。
- [ ] 库存预警、补货建议、生命周期建议和预警去重规则可复现。
