# 销售库存财务对账模块发布说明

## 发布状态

当前仅允许 Draft PR 和 Local Sandbox 验证，不代表生产发布许可。接口映射保持 `pending`，直到 JWT 联调、模块 profile、integration profile 和固定审核 HEAD CI 全部通过。

## 风险与回滚

- 本模块含财务状态、权限、迁移和不可变审计，整体按 L3 审核。
- 回滚应用提交后执行 Django 迁移回退至 `finance.0001_initial`；该回退只移除幂等键和条件唯一约束。
- 若 tenant/scope 隔离、掩码、状态冲突、审计不可变或真实连接边界失败，立即停止合并。

## 不可放行能力

- 不自动创建采购订单。
- 不执行付款、转账或提现。
- 不连接真实平台、银行或支付系统。
- 不触发真实 RPA，不使用真实凭据或真实业务数据。
