# 阶段3前端发布说明

## 发布范围

- 经营总览、销售与库存分析。
- 库存预警、补货建议、生命周期复盘和经营预警。
- 配置中心、财务只读分析和报表导出审计提示。

## 发布边界

- 阶段3新增接口当前均为 `mock`/`pending`。
- 不接入真实平台、银行或支付服务。
- 不启用自动补货、改价、清仓、上下架、资金动作或真实 RPA。
- production 保持 disabled 或 not approved。

## 发布门禁

- 前端依赖、测试、构建、API路径、安全扫描、可访问性与构建产物检查必须通过。
- 开发A接口合并后需重新完成契约、权限和运行时联调，才可更新为 `connected`。
- 当前依赖、29 项测试、生产构建、API边界和安全扫描均通过，桌面浏览器冒烟检查无控制台错误。
- 发布前待在真实 390px/窄屏浏览器环境复核移动导航抽屉和宽表滚动。
