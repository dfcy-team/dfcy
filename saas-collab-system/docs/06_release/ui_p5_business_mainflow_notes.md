# UI-P5 业务主链路发布说明

## 已完成

- 商品市调、商品主数据和采购订单使用真实 API/Mock 切换。
- 商品与采购执行动作权限、tenant 和 permission-specific data_scope。
- 供应商任务、反馈与出货执行 `tenant_id + supplier_id` 隔离。
- 列表统一分页，页面具备查询、加载、错误和空数据状态。
- 编码冻结按独立权限显示并保持幂等。
- 通用资料维护接口不能修改商品审批、商品状态或采购订单状态。
- 供应商任务完成要求完成数量与生产数量一致。
- 页码不再受 100 页限制，页大小仍最多 100。

## 保留边界

- 刊登、价格和销售订单未完成后端合同，保持 `pending/mock`。
- 不自动创建正式采购订单，不自动通知真实供应商。
- 不执行真实刊登、改价、清仓、库存修改或真实 RPA。
- 不接入真实 BigSeller、Shopee、TikTok/TK、银行或支付平台。
