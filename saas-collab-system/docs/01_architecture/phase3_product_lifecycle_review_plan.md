# 阶段3商品生命周期复盘方案

## 生命周期阶段

新品观察期、成长期、稳定销售期、滞销观察期、清仓候选、清仓中、停售、归档。

## 分析与复盘

生命周期分析使用销售趋势、毛利占位、库存覆盖、退货/异常占位、刊登状态、供应商交付和人工标记。复盘周期至少支持周和月，输出阶段建议、证据、置信度、规则版本和建议原因。

## 人工确认与审计

系统只产生状态建议。清仓、停售、归档必须由授权人员人工确认；确认、拒绝、调整和规则变更均需记录操作人、tenant、时间、原因和前后状态。

## 硬性边界

- API 和 RPA 只能提供数据。
- BI 页面不得直接改变商品状态。
- 不得自动执行改价、下架或清仓。
