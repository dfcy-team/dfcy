# 阶段3配置中心方案

## 配置项

指标阈值、预警阈值、库存安全天数、补货周期、采购交期、生命周期规则、RPA重试上限、同账号串行规则、页面变化暂停规则、数据同步频率和报表展示配置。

## 配置治理

每项配置必须定义 tenant级或系统级范围、配置版本、生效时间、修改权限、审批要求、变更记录、回滚机制和默认值。tenant 级配置不能覆盖系统级安全下限；系统级配置只由受限内部角色修改。

配置变更须记录 actor、tenant、旧值摘要、新值摘要、审批记录、版本和回滚原因。敏感配置只保存引用或脱敏元数据。

## 禁止存储

不得保存真实平台密钥、银行密码、Cookie、Session 或明文 Token。真实凭据仍由独立密钥管理和专项安全评审处理。
