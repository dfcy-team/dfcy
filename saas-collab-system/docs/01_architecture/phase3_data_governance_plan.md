# 阶段3数据治理方案

阶段3建立指标口径管理、数据来源追踪、数据更新时间、数据质量状态、缺失值处理、重复数据处理、异常值处理、tenant隔离、权限范围、财务字段脱敏、报表导出审计、指标版本管理和数据保留周期。

## 治理要求

1. 每项指标必须能追溯来源表、同步批次、计算任务、口径版本和刷新时间。
2. 缺失值按指标规则标记为 unknown、N/A 或 0，禁止静默补齐导致错误决策。
3. 重复数据按业务主键、来源事件和幂等键去重；异常值保留质量标记和人工处理入口。
4. tenant 是所有汇总、缓存、导出和预警的强制过滤条件；供应商额外按 supplier_id，财务额外按财务权限过滤。
5. 财务字段默认聚合或掩码展示；报表导出必须校验权限、记录审计并遵守保留周期。
6. 指标口径和数据保留周期变更需要版本化、审批和可回滚记录。
