## 4. 明确禁止项

1. 不在本任务中修改 backend、frontend、rpa-agent 或 `docs/04_rpa`。
2. 不提交真实 `.env`、账号、密码、Token、Cookie、Session、API Key、证书或数据库密码。
3. 不接入真实 BigSeller、Shopee、TK/TikTok、银行、支付或其他外部生产平台。
4. 不实现或启用真实自动改价、清仓、补货、上下架、刊登或财务自动对账。
5. 不允许 RPA 绕过 `/api/rpa/*`、不允许 RPA 直连数据层、不允许前端承担真实权限判断。

## 5. 下一步建议

先对 P2-A-001、P2-SEC-001 和 P2-RISK-001 完成评审结论，再按批准范围创建开发任务与分支。开发任务必须从最新 `main` 创建，且每项任务完成后提供测试、安全扫描和范围核验记录。
